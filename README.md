# RentACarProject
Bu projede CQRS ve Mediator Design Pattern ile tüm araçları filtreleme ile lokasyon ve tarih girdileri ile listeleyebilir.Admin paneli üzerinden Crud işlemlerini yapabilirsiniz.
# Kullanılan Teknolojiler
* Asp .Net Core 6.0
* MSSQL
* Code First
* CQRS
* Mediator
